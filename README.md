# Delve

**Learn anything, one tap at a time.**

Delve is a mobile app that replaces mindless scrolling with AI-powered knowledge
exploration. Pick any topic, get an interactive hierarchical breakdown, and drill
deeper into whatever catches your eye. Your explorations persist as a growing
personal knowledge base.

## How It Works

1. **Pick a topic** — type anything: "Surfing", "Nuclear Power", "The history of jazz"
2. **See the big picture** — AI generates a structured breakdown with sourced content
3. **Go deeper** — expand any section, add subtopics, branch into new areas
4. **Test yourself** — quiz mode validates what you've learned
5. **Build knowledge** — everything saves and grows over time

## Tech Stack

| Layer | Technology |
|---|---|
| Mobile | React Native + Expo, Expo Router, TypeScript |
| API | Python FastAPI on Google Cloud Run |
| AI | Claude Sonnet + web search (Anthropic API) |
| Database | Supabase (Postgres) |
| Components | Storybook for React Native |
