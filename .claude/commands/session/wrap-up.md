# Session Wrap-Up Check

Review our recent discussion and identify:

1. **Open items** - anything unresolved or needing follow-up
2. **Inconsistencies** - contradictions or unclear decisions
3. **Loose ends** - topics mentioned but not addressed
4. **Progress tracking** - if we have any issue/progress docs or task lists, are
   they up to date? Are our achievements worth documenting, or just minor side
   work?
5. **Recap** - brief summary of what was accomplished this session

If everything looks good, just confirm we're all set. Otherwise, list what's
still worth discussing.
