# Session Commands

Quick utilities for managing the current conversation session.

These are lightweight prompt wrappers - no file I/O or complex workflows, just
shorthand for common conversational checkpoints.

## Available Commands

| Command                     | Purpose                                                            |
| --------------------------- | ------------------------------------------------------------------ |
| `/session/wrap-up`          | Check if anything from the discussion is still open or unresolved  |
| `/session/improve-workflow` | Analyze a workflow gap and produce a structured improvement report |
