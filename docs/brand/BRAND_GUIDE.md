# Delve — Brand Guide

## Brand Essence

**Delve** — to reach inside something and explore it thoroughly. The name
captures exactly what the product does: you pick a topic and delve into it.

## Voice & Tone

- **Curious, not academic** — "Here's something fascinating" not "The following
  section describes"
- **Concise, not shallow** — Dense with insight, zero filler
- **Confident, not preachy** — States facts, cites sources, doesn't lecture
- **Warm, not cutesy** — Approachable without being patronizing

## Visual Direction

### Color Palette

- **Primary**: Deep indigo (`#4F46E5`) — knowledge, depth, focus
- **Surface**: Near-black (`#0F0F14`) — dark-first, easy on eyes
- **Card/Section**: Dark gray (`#1A1A24`) — subtle elevation
- **Accent**: Warm amber (`#F59E0B`) — curiosity, discovery, "new" badges
- **Text Primary**: Off-white (`#F0F0F5`)
- **Text Secondary**: Muted gray (`#8888A0`)
- **Success**: Emerald (`#10B981`) — quiz correct, version restore
- **Per-branch colors**: Each H2 branch gets a unique color from the AI,
  carried through to its children as a subtle left-border accent

### Typography

- **Headings**: System sans-serif, bold — clean and fast to render
- **Body**: System sans-serif, regular — optimized for mobile reading
- **H1**: 28pt, bold
- **H2**: 22pt, semibold
- **H3**: 18pt, medium
- **H4/Bold**: 16pt, semibold
- **Body**: 15pt, regular, 1.6 line height

### Layout Principles

- **Single-column, scrollable document** — like Notion, not like a tree widget
- **Collapsible sections** — tap heading to toggle children
- **Generous whitespace** — content breathes, not cramped
- **Left-border color accents** — 3px colored left border on each section,
  inheriting parent branch color
- **Inline action chips** — small, pill-shaped buttons below content
- **Source links** — small, muted, below content — present but not distracting

### Iconography

- Emoji-forward — each topic and node gets an AI-assigned emoji
- Minimal UI chrome — the content IS the interface
- Chevrons for collapse/expand state
- Depth indicator dots (1-4 filled circles showing current depth)

## App Store Presence

- **Tagline**: "Learn anything, one tap at a time"
- **Category**: Education
- **Keywords**: learn, explore, knowledge, AI, topics, curiosity
